        Описание

        Uber Technologies Inc.; Uber — американская международная публичная компания из Сан-Франциско, создавшая одноимённое мобильное приложение для поиска, вызова и оплаты такси или частных водителей и доставки еды. Википедия
        Дата основания: март 2009 г., Сан-Франциско, Калифорния, США
        Генеральный директор: Дара Хосровшахи (30 авг. 2017 г.–)
        Штаб-квартира: Сан-Франциско, Калифорния, США
        Доход: 11,14 миллиарда USD (2020 г.)
        Количество сотрудников: 26 900 (2019 г.)
        Основатели: Трэвис Каланик, Гарретт Кэмп
        Дочерние компании: UberEATS, Cornershop, Inc., Postmates, ЕЩЁ
